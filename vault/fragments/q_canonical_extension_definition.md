# Tentative definition of Q-canonical extension

A \(\mathcal Q\)-canonical extension of \(\mathcal A\) is a pair \((\mathcal A^\delta,\eta)\), where \(\mathcal A^\delta\) is a complete separated \(\mathcal Q\)-category and \(\eta:\mathcal A\to\mathcal A^\delta\) is a fully faithful \(\mathcal Q\)-functor. It satisfies density, meaning every object of \(\mathcal A^\delta\) can be expressed as both a weighted colimit and a weighted limit of objects from \(\mathcal A\), and compactness, meaning that whenever a formal inequality between such weighted joins and meets holds in \(\mathcal A^\delta\), it is already witnessed by a finite subconfiguration inside \(\mathcal A\).
